# PokéApp - Base de Datos Global Pokémon

Una aplicación Full-Stack interactiva construida con **React**, **Tailwind CSS**, **PokéAPI** y **Supabase**. Permite explorar la historia completa de los 1025 Pokémon existentes, filtrarlos por generación y participar en una votación global.

## 🚀 Características Principales

- **Base de Datos Completa:** Acceso a los 1025 Pokémon registrados hasta la Novena Generación.
- **Sistema de Autenticación Real:** Registro e inicio de sesión mediante **Supabase Auth**. Solo los entrenadores registrados pueden votar.
- **Votación Global Persistente:** 
  - Sistema de "Un entrenador, un voto" validado por e-mail.
  - Los votos se guardan en la nube y son visibles en tiempo real para todos los usuarios.
- **Buscador Global:** Encuentra cualquier Pokémon por nombre o ID. Cambia automáticamente al filtro "Todos" para asegurar resultados globales.
- **Filtros por Generación:** Navegación rápida entre las regiones (Gen I - Gen IX).
- **Historial de Entradas Inteligente:**
  - Muestra las descripciones oficiales en inglés de todas las versiones.
  - Agrupa entradas idénticas para una lectura limpia.
  - Oculta automáticamente generaciones previas al debut de cada Pokémon.
  - Contenedor con scroll y números romanos `sticky` para una cronología fluida.
- **Visualización de Estadísticas:** Barras de progreso con colores dinámicos según el valor (escala desde rojo oscuro hasta turquesa neón).

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Backend-as-a-Service:** Supabase (Auth & PostgreSQL).
- **API Externa:** PokéAPI (Datos e imágenes oficiales).
- **Iconos:** FontAwesome.

## 📋 Configuración de la Base de Datos (Supabase)

Para que el sistema de votos funcione, debes ejecutar el siguiente script en el **SQL Editor** de tu panel de Supabase:

```sql
-- Crear tabla de votos
create table votes (
  id bigint generated by default as identity primary key,
  email text not null unique,
  pokemon_id integer not null,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Habilitar Políticas de Seguridad (RLS)
alter table votes enable row level security;

-- Permitir que cualquiera vea los votos totales
create policy "Permitir lectura para todos" on votes
  for select using (true);

-- Permitir que usuarios autenticados inserten su voto
create policy "Permitir inserción para todos" on votes
  for insert with check (true);
```

## 📦 Instalación y Ejecución

1. Clona el repositorio e instala las dependencias:
   ```bash
   npm install
   ```
2. Ejecuta la aplicación en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Para ver la app desde otros dispositivos en tu red local:
   ```bash
   npm run dev -- --host
   ```

El proyecto, aunque funcional, quiero que sea más bonito de lo que es ahora, algunas descripciones de pokedex faltan y hay errores menores de como se ven ciertos assets. Quiero agregar nuevas funcionalidades a ser posible, como poder mostrar los sprites de cada pokemon desde la vista general, que actualmente debido a la gran carga que implica mostrar miles de imagenes de buena calidad, preferí mostrar las imagenes de pokeball de baja calidad.
