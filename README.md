# PokeVote - Plataforma Global de Entrenadores Pokémon

Una aplicación Full-Stack definitiva construida con **React 19**, **Tailwind CSS 4**, **PokéAPI** y **Supabase**. Explora la historia completa de los 1025 Pokémon, utiliza filtros avanzados, participa en la votación global de favoritos y construye tus escuadrones estratégicos.

## 🚀 Características Principales

- **Evolución Visual (Branding PokeVote):** Nueva identidad visual con tipografía redondeada, títulos neón y una interfaz de usuario oscura y moderna.
- **Base de Datos Completa con Paginación:** Visualización optimizada de los 1025 Pokémon divididos en páginas de 100 elementos para un rendimiento fluido y carga instantánea de **Official Artworks**.
- **Sistema de Filtros de Próxima Generación:**
  - **Barra Lateral de Tipos:** Filtra por hasta 2 tipos elementales simultáneamente con una interfaz de 18 botones temáticos.
  - **Generaciones Icónicas:** Botones de Gen I a Gen IX con colores representativos de sus juegos originales y efectos de hover dinámicos.
  - **Buscador Global Inteligente:** Búsqueda en tiempo real por nombre o ID que se integra con todos los demás filtros.
- **Constructor de Equipos (Team Builder):**
  - **Estrategia Personalizada:** Crea y guarda equipos de hasta 6 Pokémon.
  - **Ajustes de Combate:** Configura motes personalizados, selecciona habilidades y define el set de 4 movimientos para cada integrante.
  - **Buscador Técnico de Ataques:** Interfaz de selección de movimientos con datos en tiempo real de Tipo, Poder, Precisión y Categoría.
  - **Gestión Persistente:** Modifica, actualiza o elimina tus escuadrones guardados en cualquier momento.
- **Wiki Interconectada:**
  - **Enciclopedia de Movimientos:** Explora todos los ataques, sus efectos detallados y qué Pokémon pueden aprenderlos.
  - **Base de Habilidades:** Consulta los efectos de cada habilidad y descubre qué especies las poseen.
- **Sistema de Autenticación Real:** Registro e inicio de sesión seguro gestionado por **Supabase Auth**.
- **Votación Global Persistente:** 
  - Regla de "Un entrenador, un voto" vinculada a la cuenta de correo.
  - Contador de votos global en tiempo real para cada Pokémon.
  - **Ranking Dinámico:** Sección especial con la "King's Rock" (Roca del Rey) para visualizar a los Pokémon más votados por la comunidad.
- **Vista de Detalles Expandida:**
  - **Arsenal de Movimientos:** Paginación de ataques con estadísticas completas.
  - **Historial de Entradas:** Cronología completa de descripciones oficiales (en inglés) agrupadas por generación.
  - **Estadísticas Base:** Barras de progreso con colores dinámicos (de Rojo Crítico a Turquesa Neón) según el valor de la estadística.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Backend-as-a-Service:** Supabase (Auth & PostgreSQL).
- **API Externa:** PokéAPI.
- **Iconografía:** FontAwesome.

## 📋 Configuración de la Base de Datos (Supabase)

Para que el sistema completo funcione, ejecuta este script en el **SQL Editor** de Supabase:

```sql
-- Tabla de Votos
create table votes (
  id bigint generated by default as identity primary key,
  email text not null unique,
  pokemon_id integer not null,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Tabla de Equipos
create table teams (
  id bigint generated by default as identity primary key,
  user_email text not null,
  team_name text,
  members jsonb not null, -- Almacena array de pokémon, motes, ataques y habilidades
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Seguridad (RLS)
alter table votes enable row level security;
alter table teams enable row level security;

-- Políticas
create policy "public_select_votes" on votes for select using (true);
create policy "public_insert_votes" on votes for insert with check (true);

create policy "Users can manage their own teams" on teams
  for all using (auth.jwt() ->> 'email' = user_email)
  with check (auth.jwt() ->> 'email' = user_email);
```
